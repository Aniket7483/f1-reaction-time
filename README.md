# 🏎️ F1 Reaction Time Game

An F1-inspired reaction time web game built using **HTML, CSS, and JavaScript**.

Test your reflexes against Formula 1 start lights and discover which driver personality matches your reaction speed.

---

## 🚦 How It Works

1. Click **Start Race**
2. Red lights will appear one by one
3. After a random delay, the lights go out
4. Click as fast as possible
5. Get your reaction time + F1 driver personality

---

## 🏁 Features

- Realistic F1-style start light sequence
- Random delay for fairness
- Reaction time calculation (ms)
- Best time tracking using `localStorage`
- Driver personality reveal:
  - 🏎️ **Max Verstappen** (< 200ms)
  - 🎵 **Carlos Sainz** (200–299ms)
  - 🐔 **Charles Leclerc** (300–399ms)
  - 🛞 **Rookie Driver** (400ms+)
- False start detection
- Smooth animations
- Fully responsive design

---

## 🧠 Tech Stack

- HTML5
- CSS3 (Animations + Transitions)
- Vanilla JavaScript (No libraries)
